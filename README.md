## Run Your Local

To clone this project:
```bash
  git clone https://github.com/aleyaktas/shoppingLandingPage.git
```
To make necessary downloads for project:
```bash
  cd shoppingLandingPage
  yarn install
```
To run project:
```bash 
  yarn start
```
